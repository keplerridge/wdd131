Kepler Ridge

WDD 131-03

Overview

Purpose

This site will provide a more accessible and user friendly experience for quickly finding book reviews with a friendly home page, search system, and categorizations.

Audience

The audience is the more casual book reader. There are plenty of options for the avid book reader but this will provide a better experience for the more casual readers.

Dynamic elements

There will be multiple different categories that will show different books in different list sizes and different layouts depending on screen size. There will then be a profile subpage. There will be basic login functionality, depending on how everything else goes, I will try to work in legitimate hashing and login security with a small database to hold profile information.

Content

Home page

This page will hold several different book review lists broken apart by genres and it will show highly rated books. It will also have a search bar so that you can find books that you want. It will have the option to create a new review on a book as well as enter a new book to the review process. Ideally I wil have a database setup to hold all of this information.

Your Library

This will hold all of the books associated with this users library. It will have an option to remove books from the library. It will also have a search bar in there to search the library. Books will be added from the home page not here.

Profile

This page will hold profile information such as username, email, profile picture(I may just use a generic male or female image for the time being). It will also have the option to update and edit the profile.
